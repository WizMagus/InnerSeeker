# InnerSeeker

本專案為課堂練習，以偉特塔羅的單張牌占卜為主題，基於MVC+DAO架構撰寫，主要訓練內容如下：
1. MVC分層結構
2. MySQL CRUD
3. 使用者登入流程

## 功能概覽
- **登入 / 註冊**
  - 登入：帳密驗證（密碼暫時於資料庫以明碼呈現）
  - 註冊：帳號、密碼、密碼確認、提示/提示答案
  - 訪客登入：可占卜但不寫入歷史紀錄
- **占卜流程**
  - 輸入問題 → 洗牌（含簡易動畫）→ 抽牌（支援正位/逆位）→ 顯示牌面與 prompt
- **歷史紀錄（DB）**
  - 會員抽牌才會寫入 `draw_history`
  - 牌組資料從 `rws_tarot` 讀取
